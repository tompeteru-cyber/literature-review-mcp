# Literature Review MCP Server - Output Directory

This directory contains all generated files from the Literature Review MCP Server operations.

## Directory Structure

- **analyses/** - Tier 1, 2, 3 filtering results and AHP analyses
- **chapters/** - Generated literature review chapters in various formats
- **search-results/** - Database search results and cached queries
- **methodologies/** - Extracted methodology analyses from papers
- **citations/** - Citation network analysis results and visualizations
- **trends/** - Research trend analysis over time
- **reports/** - Comprehensive analysis reports and summaries

## File Naming Convention

Files are automatically named with timestamps and descriptive names:
- `YYYY-MM-DD_HH-MM-SS_operation-name.format`
- Example: `2025-09-15_18-30-45_tier1-filtering.json`

## Generated File Types

- **JSON** - Structured data and analysis results
- **Markdown** - Literature review chapters and reports
- **LaTeX** - Academic paper format outputs
- **CSV** - Tabular data exports
- **HTML** - Web-viewable reports with visualizations
